# Veröffentlichung auf GitHub

1. Öffentliche Repository-Seite anlegen, empfohlen: `carbor-media-library`.
2. Dieses Quellprojekt in den lokalen Carbor-Ordner kopieren.
3. `scripts/check_secrets.py` ausführen.
4. `scripts/init_git.sh` ausführen.
5. GitHub-Repository als Ziel eintragen und `main` hochladen.
6. Screenshots ergänzen, die keine privaten Dateinamen oder Pfade zeigen.
7. Zunächst einen Beta-Release erstellen, z. B. `v1.0.3-beta.1`.
8. Release-Archiv und SHA-256-Prüfsumme anhängen.
9. Später DEB/AppImage nur nach realem Installationstest anhängen.

Vor der öffentlichen Veröffentlichung sollte außerdem geprüft werden, ob der Name „Carbor“ markenrechtlich konfliktfrei nutzbar ist. Diese Projektdateien ersetzen keine Rechtsberatung.
