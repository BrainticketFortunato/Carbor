# Release-Checkliste

- [ ] Versionsnummer in Programm, Hilfe und Changelog identisch
- [ ] Syntaxprüfung erfolgreich
- [ ] keine API-Schlüssel oder privaten Pfade enthalten
- [ ] Hilfe aktuell und vollständig
- [ ] Drittanbieterhinweise aktuell
- [ ] Icon und Desktop-Starter geprüft
- [ ] Start auf frischem Linux-System getestet
- [ ] Ein-Instanz-Sperre getestet
- [ ] VLC-Fortschritt getestet
- [ ] Release-Archiv und SHA-256 erzeugt
- [ ] Release als Beta markieren, solange kein Fremdtest erfolgt ist
